mod_rewrite는 php 코드가 실행되기 전, 웹서버 apache에 적용되는 것
mod_rewrite은 URL을 변환(Rewrite) 하는 기능을 제공하는 모듈이며, 사용자가 요청한 URL을 서버 내부에서 다른 경로로 바꿔주는 역할을 담당함

- 🔗 예쁜 URL (Pretty URL): /post/hello-world 같은 깔끔한 주소 만들기
- 🚀 SEO 최적화: 검색엔진이 읽기 좋은 URL로 개선
- 🔄 리디렉션 (Redirect): 옛날 URL을 새 URL로 자동 이동 (예: 도메인 변경 시)
- 🔐 보안/접근제어: 특정 파일 확장자 접근 막기 등
- 🧭 경로 정리: 내부 디렉토리 구조를 숨기거나 단순화

용례)
- 'www' 없이 naver.com 요청이 오면, 요청 url 맨앞에 자동으로 'www.'를 붙여줌
- 특정 파일 및 확장자 요청 차단 (서버 열어놓고 로그 찍어보면 해킹봇 요청이 엄청 많은데, 이것들을 차단할 수 있음. 하지만 그만큼 정교하고 수많은 정규식 및 컨디션 구현 필요)
